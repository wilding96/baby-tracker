# 纯代码编辑模式

当该命令被触发或此规则生效时，需要遵循以下说明，严禁越权操作：

禁止执行命令：绝对不要尝试调用任何终端工具或执行 Shell 命令。包括但不限于 npm、pnpm、yarn、node、cmd 或 powershell。不要尝试安装依赖、启动开发服务器或进行编译。

仅限文件操作：你唯一被允许的行为是读取现有代码（Read）、分析逻辑、以及直接修改或创建代码文件（Write）。

环境静默假设：默认本地开发环境、Node.js 版本（nvm）和项目依赖已经由用户手动配置完成。如果发现代码中有报错或缺失依赖，只需在代码层面进行修正或在回复中提醒用户，不得擅自尝试通过命令解决。

专注业务逻辑：直接针对 Vue 组件、脚本逻辑或配置文件进行修改。完成代码编写后直接保存并停止，不要询问是否需要运行测试或部署。
