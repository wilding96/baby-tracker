# 《雷电Like Roguelite STG》
# 武器、子弹、僚机成长系统设计文档 V1.0

---

# 一、设计目标

## 当前问题

原系统：

```
敌人死亡
 ↓
掉落道具
 ↓
玩家碰到
 ↓
武器+1
```

问题：

1. 玩家容易漏吃
2. 玩家不知道自己获得什么
3. 升级没有仪式感
4. Build没有形成
5. 僚机成长没有记忆点

---

# 新设计目标

玩家每局产生：

> "我正在打造一台属于自己的战机"

核心体验：

```
战斗
 ↓
收集能源
 ↓
选择成长方向
 ↓
飞机视觉变化
 ↓
形成Build
 ↓
挑战Boss
```
