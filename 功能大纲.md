## 4. 核心功能模块

### 4.1 首页仪表盘 (`/`)

- 展示宝宝姓名 + 当前日期
- 成长事件总数 & 即将到来的事件数
- 下一个事件的倒计时（天数）
- **成长时间线**：按时间轴展示 体检/疫苗/大事件/其他 四类事件，支持删除
- **快捷入口卡片**：记录、统计、情绪日记、留言板、三消游戏、雷电游戏
- **Modal 弹窗添加成长事件**：标题、日期、类型(下拉)、备注

### 4.2 喂养/睡眠/换尿布记录 (`/record`)

- **喂养**：切换 配方奶↔母乳，配方奶带 ml 滑块 + 快速预设(60/90/120/150/180)，记录开始时间
- **睡眠**：开始/结束时间选择，自动计算并显示时长
- **换尿布**：记录类型(湿/臭/混合) + 开始时间

### 4.3 数据分析 (`/stats`)

- 近 7 天数据图表（Recharts）
- **奶量 Tab**：柱状图展示每日总 ml，附日均奶量提示
- **睡眠 Tab**：面积图展示每日总睡眠小时数
- **尿布 Tab**：堆叠柱状图(湿/臭/混合) + 数量汇总卡片

### 4.4 留言板 (`/board`)

- 公开访问，无需登录
- 输入昵称(可选) + 留言内容
- 彩色交替卡片展示(黄/绿/蓝/桃)，最新在前

### 4.5 情绪日记 + Live Photo (`/mood`)

- **三种情绪模式**，通过 CSS class 切换：

  | 模式 | 情感定位 | 背景 | 文字 | 视频录制动效 |
  |------|----------|------|------|-------------|
  | 褪色 (faded) | 疲惫但温柔 | 暖灰渐变 `#e4e3df→#f0efe9` | `#6a6a6a` 宽松字距 | 静止 |
  | 霓虹 (neon) | 压抑/窒息 | 深蓝→紫→红渐变，8s 呼吸动画 | 蓝紫色 + 红蓝 offset text-shadow | 正弦波漂浮 |
  | 焦躁 (fever) | 焦虑/崩溃 | 暗红黑径向渐变 `#3a0e05→#0a0100` | 暗暖色 + 红色偏移 + 0.15s 抖动 | 随机剧烈抖动 |

- 通用层：SVG feTurbulence 噪点纹理叠加（质感）
- 支持编辑文案
- **Live Photo 导出**：DOM → Canvas 截图 → 30fps 视频录制 → 服务端 ffmpeg 转码 → 写入 Apple ContentIdentifier 元数据 → 打包 ZIP 下载

### 4.6 设置中心 (`/settings`)

- 宝宝资料卡片（点击跳转编辑）
- **家庭共享**：展示邀请码，支持复制，用于邀请其他家庭成员
- **通用设置**：推送通知开关、深色模式(占位)、PWA 安装引导
- **数据与支持**：导出数据(Excel 占位)、帮助与反馈、隐私政策
- 退出登录（带确认）

### 4.7 宝宝资料编辑 (`/settings/profile`)

- 编辑宝宝姓名、生日、性别
- 保存到 Supabase `babies` 表

### 4.8 三消游戏 (`/game`)

- 8×8 棋盘，6 种糖果类型（emoji：苹果/橙子/柠檬/葡萄/蓝莓/草莓）
- 相邻交换逻辑 + 连锁消除 + 连击倍率
- 分数 & 步数计数器
- 每隔 500 分弹窗提交排行榜里程碑
- 入口链接到排行榜页面

### 4.9 三消排行榜 (`/game/leaderboard`)

- 前 50 名，按分数降序、步数升序排列
- 前三名冠亚季军 emoji 奖牌

### 4.10 雷电射击游戏 (`/game/raiden`)

- 全功能 Canvas 纵版卷轴射击游戏，约 3500 行
- **操作**：WASD/方向键移动，自动射击，空格放炸弹
- **敌人类**：迷你Boss、轰炸机
- **Boss 类**：堡垒型、航母型、眼睛型（3 种）
- **系统**：金币、道具、开局画面、HUD（生命/炸弹/金币/分数/Boss 警告）
- **暂停**：Esc 键
- **音效**：Web Audio API 芯片音乐风格 BGM + 音效（`useGameAudio` hook）

### 4.11 旧版首页 (`/legacy`)

- 今日喂养/睡眠/尿布汇总
- 支持编辑和删除（弹窗形式）
- 不同 UI 风格（旧版保留）

## 5. API 路由

### `POST /api/generate-live-photo`

- 接收 FormData：`image` (JPG) + `video` (WebM)
- 服务端 `fluent-ffmpeg` 将 WebM 转码为 MOV (H.264 + AAC)
- `exiftool-vendored` 写入 Apple Live Photo 元数据（ContentIdentifier, StillImageTime）
- 打包 ZIP 返回下载

## 6. 后端数据表 (Supabase)

| 表名 | 用途 | 主要字段 |
|------|------|---------|
| `babies` | 宝宝档案 | id, name, birthday, gender, invite_code |
| `baby_users` | 用户-宝宝关联 | user_id, baby_id, role(owner/member) |
| `growth_events` | 成长事件 | baby_id, title, event_date, type, notes, created_by |
| `logs` | 护理记录 | baby_id, type(feeding/sleep/diaper), start_time, end_time, details(JSON) |
| `message_board_messages` | 留言 | nickname, content, created_at |
| `game_leaderboard` | 排行榜 | nickname, score, moves, created_at |
